Untuk melihat diagram langsung di VS Code:

1. Install PlantUML Extension

  1. Buka VS Code Extensions (Ctrl+Shift+X)
  2. Search: PlantUML
  3. Install extension oleh jebbs

2. Install Graphviz (Required)

# Windows (dengan Chocolatey)

# Atau download installer dari:
# https://graphviz.org/download/

3. View Diagrams

  1. Buka file .puml di VS Code
  2. Press Alt+D (Windows/Linux) atau Option+D (Mac)
  3. Diagram akan ditampilkan di panel samping

Untuk export diagrams ke format gambar:

Using PlantUML Command Line:

# Download PlantUML JAR from:
# https://github.com/plantuml/plantuml/releases

java -jar plantuml.jar diagrams/*.puml

# Generate PDF
java -jar plantuml.jar -tpdf diagrams/class_diagram.puml
